## Cara Akses Materi

1. Clone repository:
    ```bash
    git clone https://github.com/username/nama-repo.git
    ```
2. Checkout ke branch pertemuan yang diinginkan, misalnya:
    ```bash
    git checkout pertemuan-1
    ```
3. Buka file materi di folder yang tersedia.
